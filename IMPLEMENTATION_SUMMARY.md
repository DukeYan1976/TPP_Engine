## 使用流程

### 基本流程（世界坐标系）
1. 选择STEP文件
2. 点击选择一个面
3. 设置刀路参数
4. 点击"计算刀路"
5. 点击"输出刀路"
6. 选择保存位置

### 高级流程（自定义坐标系）
1. 选择STEP文件
2. 选择"自定义坐标系"单选按钮
3. 点击"设置坐标系"
4. 依次点击三个点：
   - 原点
   - X轴方向点
   - XY平面点
5. 查看坐标系可视化（红X绿Y蓝Z）
6. 点击选择一个面
7. 计算刀路
8. 输出刀路（坐标已转换）

## 技术亮点

1. **零拷贝数据传输**：遵循项目规范，使用 bytes 字段
2. **法线计算**：使用 D1() 获取切向量，叉乘得法线
3. **坐标转换**：4x4齐次变换矩阵，numpy高效计算
4. **UI集成**：无缝集成到现有GUI，不破坏原有功能
5. **可视化**：坐标系三色轴显示，直观清晰
