# 坐标系功能使用说明

## 新增功能

### 1. 世界坐标系显示
- **自动显示**：加载STEP文件后，自动在原点(0,0,0)显示世界坐标系
- **视觉特征**：
  - X轴：红色
  - Y轴：绿色  
  - Z轴：蓝色
  - 半透明显示（透明度40%）
  - 轴长度为模型包围盒对角线的15%
- **固定参考**：始终可见，不参与选择交互

### 2. 自定义工件坐标系
- **设置方式**：三点拾取定义
  1. 点击原点位置
  2. 点击X轴正方向
  3. 点击XY平面上一点（定义Z轴方向）
- **视觉特征**：
  - 不透明显示（更明显）
  - 轴长度为模型包围盒对角线的20%（比世界坐标系大）
- **变换信息显示**：
  - 原点坐标 (X, Y, Z)
  - 旋转角度 (绕X/Y/Z轴的欧拉角)

### 3. 坐标系调整功能
- **调整按钮**：设置自定义坐标系后启用
- **调整方式**：重新三点拾取（保留世界坐标系作为参考）
- **实时更新**：调整后立即更新变换信息

## 使用流程

### 基本流程
1. 加载STEP文件 → 自动显示世界坐标系
2. 选择"自定义坐标系"单选按钮
3. 点击"设置"按钮
4. 依次点击三个点定义坐标系
5. 查看变换信息（原点坐标和旋转角度）

### 调整流程
1. 在自定义坐标系模式下，点击"调整"按钮
2. 世界坐标系保持显示作为参考
3. 重新拾取三个点
4. 自定义坐标系更新到新位置

### 输出刀路
- 世界坐标系模式：刀路点直接输出世界坐标
- 自定义坐标系模式：刀路点自动转换到工件坐标系

## 技术实现

### 代码增加量
- 新增函数：3个（约150行）
  - `display_world_coordinate_system()` - 显示世界坐标系
  - `display_custom_wcs()` - 显示自定义坐标系
  - `update_wcs_info()` - 更新变换信息
  - `start_wcs_adjustment()` - 启动调整模式
- UI控件：2个按钮 + 1个信息标签（约30行）
- 修改现有函数：5处（约20行）

### 关键技术
- 使用 `AIS_Trihedron` + `Geom_Axis2Placement` 显示坐标系
- 使用 `gp_Ax2` 定义坐标系位置和方向
- 欧拉角分解：从旋转矩阵提取ZYX顺序的旋转角度
- 坐标变换：4x4齐次变换矩阵

## 故障排查

### 坐标系不显示
- 检查是否已加载STEP文件
- 检查控制台是否有错误信息
- 尝试重新加载文件

### 坐标系太小/太大
- 轴长度自动根据模型大小计算
- 世界坐标系：包围盒对角线的15%
- 自定义坐标系：包围盒对角线的20%

### 点击拾取无响应
- 确保点击在模型表面或顶点上
- 查看状态栏提示信息
- 如果卡住，切换回世界坐标系模式重置
