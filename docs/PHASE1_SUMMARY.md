## 技术架构

### 通信层
```
Python GUI → gRPC → C++ Server → Mock CAM DLL
     ↑                                  ↓
     └──────── bytes数据传输 ───────────┘
```

### 数据传输
- 遵循零拷贝原则
- 使用 `bytes` 字段传输几何数据
- numpy `tobytes()` / `frombuffer()` 转换

---

## 核心参数

**默认设置**:
- 刀路模式: 行切
- 起始方向: V向
- 刀路数: 10

**参数范围**:
- 刀路数: 3-20（滑块调节）
- 方向: U向/V向
- 模式: 行切/环切

---

## 使用方法

### 启动服务器
```bash
cd cpp_server/build
./cam_server
```

### 启动GUI
```bash
cd python_client
python gui_client.py
```

### 操作流程
1. 选择STEP文件
2. 点击3D视图选择面
3. 设置刀轨参数
4. 计算刀路
5. 查看结果
