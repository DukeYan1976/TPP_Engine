# 刀路输出功能实施说明

## 功能概述

本次更新添加了以下功能：
1. **工件坐标系选择**：支持世界坐标系和自定义坐标系
2. **刀路输出**：将计算的刀轨点和法线输出到文本文件
3. **法线数据**：刀轨计算现在同时返回每个点的表面法线（单位刀轴矢量）

## 修改文件清单

### 1. Protocol Buffers
- `cam_service.proto`: 添加 `raw_normals` 字段到响应消息

### 2. C++ 后端
- `mock_cam/mock_cam.h`: 添加法线输出参数
- `mock_cam/mock_cam.cpp`: 
  - 使用 `D1()` 替代 `D0()` 计算点和切向量
  - 计算法线 = du × dv 并归一化
  - 行切和环切模式都支持法线计算
- `cpp_server/server.cpp`: 接收和发送法线数据

### 3. Python 客户端
- `python_client/cam_calculator.py`: 返回 (points, normals) 元组
- `python_client/gui_client.py`:
  - 添加工件坐标系UI组件
  - 实现三点拾取定义坐标系
  - 坐标系可视化（XYZ轴）
  - 刀路输出功能
  - 坐标转换（点和法线）

## 使用方法

### 1. 重新编译

```bash
cd /Users/y/openclaw/workspace/myGHDevelop/TPP_Engine
./rebuild.sh
```

### 2. 启动服务端

```bash
cd cpp_server/build
./cam_server
```

### 3. 启动客户端

```bash
cd python_client
python3 gui_client.py
```

### 4. 使用工件坐标系

#### 世界坐标系（默认）
- 直接选择"世界坐标系"单选按钮
- 输出的刀路将使用模型的原始坐标

#### 自定义坐标系
1. 选择"自定义坐标系"单选按钮
2. 点击"设置坐标系"按钮
3. 依次在3D视图中点击三个点：
   - **第1点**：工件坐标系原点
   - **第2点**：X轴正方向上的点
   - **第3点**：XY平面上的点（用于确定Z轴方向）
4. 系统自动计算并显示坐标系（红色=X，绿色=Y，蓝色=Z）

### 5. 输出刀路

1. 计算刀路完成后，"输出刀路"按钮会激活
2. 点击"输出刀路"按钮
3. 选择保存位置（默认：~/Documents）
4. 文件格式：

```
; Toolpath Output
; Coordinate System: World/Custom
; Total Points: 1234
; Path Length: 567.890 mm
; Generated: 2026-03-02 10:43:29
;
; Format: X Y Z Nx Ny Nz
;
0.000000 0.000000 10.000000 0.000000 0.000000 1.000000
0.500000 0.000000 9.950000 0.000000 0.000000 1.000000
...
```

## 输出文件格式说明

- **注释行**：以 `;` 开头
- **数据行**：6个浮点数，空格分隔
  - `X Y Z`: 刀位点坐标（在选定的工件坐标系下）
  - `Nx Ny Nz`: 单位法线向量（刀轴方向）

## 坐标转换说明

### 点坐标转换
使用4x4齐次变换矩阵：
```
P_wcs = T * P_world
```

### 法线转换
仅使用旋转部分（3x3）：
```
N_wcs = R * N_world
```

## 技术细节

### 法线计算
- 使用 OpenCASCADE 的 `GeomAdaptor_Surface::D1(u, v, pnt, du, dv)`
- 法线 = `du.Crossed(dv)` 然后归一化
- 确保法线长度为1（单位向量）

### 数据传输
- 遵循项目规范：使用 `bytes` 字段传输原生数组
- 法线数据与顶点数据分离，但点数相同
- Python 端使用 `np.frombuffer()` 零拷贝解析

### 工件坐标系
- 使用右手坐标系
- Z轴 = X × Y（临时）
- Y轴 = Z × X（确保正交）
- 变换矩阵存储为 numpy 4x4 数组

## 已知限制

1. 工件坐标系拾取使用面中心点，不是精确的点击位置
2. 坐标系轴长度固定为50mm（可根据模型大小调整）
3. 输出格式为简单文本，后续可扩展为其他格式（CSV, G代码等）
