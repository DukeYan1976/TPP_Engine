**日期**: 2026-02-28  
**版本**: v1.2

---

## 实施方案

采用**方案1（简化版）**，提供自然的交互体验。

---

## 功能特性

### 1. 面选择（始终启用）

**交互方式**:
- 点击面 → 直接选中并高亮（蓝色半透明）
- 拖拽 → 旋转视图
- 滚轮 → 缩放视图
- 重新点击 → 切换到新选择的面

**优势**:
- 无需模式切换
- 符合CAD软件习惯
- 操作流畅自然

### 2. 显示模式控制

**UI位置**: 右侧控制面板 - "显示模式"组

**按钮**:
- [线框] - 切换到线框模式
- [着色] - 切换到着色模式（默认）

**功能**:
```python
线框模式:
- 显示模型边界线
- 适合查看拓扑结构
- 方法: SetModeWireFrame()

着色模式:
- 显示实体渲染
- 适合查看几何形状
- 方法: SetModeShaded()
```

---

## 技术实现

### 显示模式切换

```python
def set_wireframe_mode(self):
    self.viewer._display.SetModeWireFrame()
    
def set_shaded_mode(self):
    self.viewer._display.SetModeShaded()
```

### 面选择模式

```python
# 初始化时设置为面选择模式
self.viewer._display.SetSelectionModeFace()

# 注册选择回调（始终启用）
self.viewer._display.register_select_callback(self.on_face_selected)
```

## 使用方法

### 选择面
1. 加载STEP文件
2. 直接点击3D视图中的面
3. 面自动高亮（蓝色）
4. 显示面积和UV范围
5. 可重新点击选择其他面

### 切换显示模式
1. 点击"线框"按钮 → 查看边界线
2. 点击"着色"按钮 → 查看实体渲染
3. 状态栏显示当前模式

## 与原需求的对比

| 项目 | 原需求 | 实施方案 | 优势 |
|------|--------|----------|------|
| 面选择 | 需要"选择面"按钮切换模式 | 始终启用，点击即选 | 更自然，无需切换 |
| 显示模式 | 右键菜单 | 工具栏按钮 | 更直观，不冲突 |
| 模式数量 | 3种（线框/着色/带边） | 2种（线框/着色） | 简化，够用 |
