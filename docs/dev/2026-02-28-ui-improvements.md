# UI改进 - 显示模式优化

**日期**: 2026-02-28  
**版本**: v1.3

---

## 修改内容

### 1. 显示模式位置调整

**修改前**:
- 位置：面选择区下方
- 与参数控制混在一起

**修改后**:
- 位置：控制面板最底部
- 添加分隔线明确区分
- 独立的"显示模式"组

**布局**:
```
┌─────────────────┐
│ 文件选择        │
├─────────────────┤
│ 面选择          │
├─────────────────┤
│ 刀轨参数        │
├─────────────────┤
│ 计算按钮        │
├─────────────────┤
│ 状态/点数       │
├─────────────────┤
│ (弹性空间)      │
├═════════════════┤ ← 分隔线
│ 显示模式        │
│ ○ 着色          │
│ ○ 线框          │
└─────────────────┘
```

### 2. 显示模式改为单选

**修改前**:
- 两个独立按钮
- 可能同时按下

**修改后**:
- 单选按钮组（QRadioButton）
- 互斥选择
- 默认选中"着色"

**代码**:
```python
self.display_mode_group = QButtonGroup()
self.radio_shaded = QRadioButton("着色")
self.radio_wireframe = QRadioButton("线框")
self.radio_shaded.setChecked(True)  # 默认着色
```

### 3. 模型颜色优化

**修改前**:
- 灰色 (0.7, 0.7, 0.7)
- 半透明 (transparency=0.5)
- 不够醒目

**修改后**:
- 橙色 (1.0, 0.6, 0.2)
- 不透明 (transparency=0.0)
- 醒目且专业

**选中面高亮**:
- 蓝色 (0.2, 0.5, 1.0)
- 半透明 (transparency=0.3)
- 与模型颜色对比明显

---

## 颜色方案

| 元素 | 颜色 | RGB | 透明度 | 用途 |
|------|------|-----|--------|------|
| 模型 | 橙色 | (1.0, 0.6, 0.2) | 0.0 | 醒目显示零件 |
| 选中面 | 蓝色 | (0.2, 0.5, 1.0) | 0.3 | 高亮选择 |
| 刀轨 | 绿色 | (0.0, 0.8, 0.0) | 0.0 | 加工路径 |
| 背景 | 浅灰 | (0.95, 0.95, 0.95) | - | 中性背景 |
