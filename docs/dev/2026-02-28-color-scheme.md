## 面选择修复

### 问题
多面STEP文件中，总是默认选择最大面，无法重新选择其他面。

### 原因
代码中有"最大面后备逻辑"：
```python
# 错误逻辑
if not selected_face:
    # 回退到选择最大面
    for face in self.faces:
        if area > max_area:
            selected_face = face  # 总是选最大面
```

### 修复
移除后备逻辑，只选择用户点击的面：
```python
# 正确逻辑
try:
    selected_face = topods.Face(selected_shape)
except:
    for face in self.faces:
        if face.IsSame(selected_shape):
            selected_face = face
            break

# 如果没找到，直接返回（不选择）
if not selected_face:
    return
```
